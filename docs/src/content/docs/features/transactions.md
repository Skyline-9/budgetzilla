Transactions are the core of Budgetzilla. Every purchase, payment, or income you record becomes a transaction.

## Adding Transactions

### Quick Add

Press `N` anywhere in the app to open the quick add dialog. Fill in:

- **Amount** — The transaction amount (positive for income, negative for expenses)
- **Description** — What was this transaction for?
- **Category** — Select or create a category
- **Date** — Defaults to today

### From the Transactions Page

Navigate to the Transactions page and click the **+ Add Transaction** button for more detailed entry.

## Transaction Fields

| Field | Description |
|-------|-------------|
| Amount | The monetary value (positive = income, negative = expense) |
| Description | A brief note about the transaction |
| Category | The spending/income category |
| Date | When the transaction occurred |
| Notes | Optional additional details |

## Searching Transactions

Use **Cmd/Ctrl + K** to open global search. You can search by:

- Description text
- Category name
- Amount
- Date range

## Filtering and Sorting

On the Transactions page, you can:

- Filter by date range
- Filter by category
- Filter by income/expense type
- Sort by date, amount, or category

## Editing Transactions

Click on any transaction to open the edit dialog. All fields can be modified.

## Deleting Transactions

In the edit dialog, click **Delete** to remove a transaction. This action cannot be undone (unless you have a backup).

## Bulk Operations

Select multiple transactions using checkboxes to:

- Delete multiple transactions
- Change category for multiple transactions
- Export selected transactions
