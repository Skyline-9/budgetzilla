title: Keyboard Shortcuts
description: Quick reference for all keyboard shortcuts.
---

Budgetzilla supports keyboard shortcuts for common actions.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open global search |
| `N` | Add new transaction |
| `Esc` | Close dialog/modal |

## Navigation

| Shortcut | Action |
|----------|--------|
| `G` then `D` | Go to Dashboard |
| `G` then `T` | Go to Transactions |
| `G` then `C` | Go to Categories |
| `G` then `S` | Go to Settings |

## Transaction List

| Shortcut | Action |
|----------|--------|
| `J` | Move to next transaction |
| `K` | Move to previous transaction |
| `Enter` | Open selected transaction |
| `Delete` | Delete selected transaction |

## Dialogs

| Shortcut | Action |
|----------|--------|
| `Esc` | Cancel and close |
| `Cmd/Ctrl + Enter` | Save and close |
| `Tab` | Move to next field |
| `Shift + Tab` | Move to previous field |

## Desktop App Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd + ,` | Open Settings |
| `Cmd + Q` | Quit application |
| `Cmd + W` | Close window |
| `Cmd + M` | Minimize window |
