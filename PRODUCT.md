# Product

## Register

product

## Users
Privacy-conscious individuals who want to track their money without sending sensitive financial data to the cloud. They value speed, offline usability, and high-end software craftsmanship.

## Product Purpose
A browser-native, local-first personal finance tracker that runs entirely on-device (SQLite WASM + local Gemma 4 receipt scanning). It aims to offer the speed and premium feel of a native desktop app while guaranteeing 100% data privacy.

## Brand Personality
- Tactile: Elements feel like physical, precision-milled buttons and panels.
- Refined: Minimal decoration, but maximum care in details (shadows, fonts, micro-animations).
- Empowering: Fast, clear, and focused entirely on the user's data and workflows.

## Anti-references
- Flat/Sterile SaaS: Monochromatic dark gray interfaces with zero depth, no borders, and no tactile feedback.
- Visually Loud Dashboards: Heavy gradients, non-standard layout blocks, and crowded visual noise.
- Sluggish Web App: Slow load states, layout shifts, or missing hover indicators.

## Design Principles
1. **Tactile Elevation**: Leverage subtle borders, wide diffused shadows, and glassmorphic layers to make panels feel like real objects floating over a base canvas.
2. **Interactive Delight**: Every hover and active state should provide feedback (micro-scaling, background brightening, border-color shifts) to make the app feel responsive and alive.
3. **Data Clarity**: Present complex financial metrics with premium typography (Space Grotesk + Inter) and responsive layouts that adjust seamlessly to screen sizes.
4. **Local Sovereignty**: Interface details should feel secure and private—no cloud loading icons, instant page transitions, and stable desktop/mobile packaging.

## Accessibility & Inclusion
- Contrast: Ensure a minimum 4.5:1 ratio for text and interactive UI elements.
- Keyboard Friendliness: All interactive controls are focusable and support keyboard shortcuts.
- Reduced Motion: Respect `prefers-reduced-motion` media queries with simplified crossfades or instant states.
